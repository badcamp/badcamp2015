Case Study: Large-Scale, Server Side Clustering for Mapping with the Leaflet-Geocluster Stack

How do we use Drupal and current mapping technology to provide a usable, responsive map application that can scale up to the order of 10k data points?

In this case study, we'll walk through an actual client build of a dense-point data map that utilizes Leaflet and Geocluster to implement server side clustering forÂ mapping with Leaflet for beautiful, scalable, and responsive mapping applications. The first take-away from this session will be to gain an understanding of how to implement a Leaflet-based application on top of Geocluster to create a powerful, responsive, user-friendly (and pretty!) map completely in Drupal 7.

Secondly, we'll take a look at what it's going to take to port this solution to Drupal 8. The architecture has a deep opportunity for growth, but Drupal 8 is on the horizon, and we should innovate there. Lets put some fire behind this so we can keep mapping applications moving forward with the rest of the community.
